# Fangwen Zhao's Personal Website

A minimal academic website built with Jekyll for GitHub Pages.

## Deployment

1. Create a new GitHub repository named `fwzhao.github.io`
2. Upload all files from this folder to the repository
3. Replace `photo.jpg` with actual photo
4. Go to Settings → Pages → Enable GitHub Pages (source: main branch, root folder)
5. Site will be live at https://fwzhao.github.io

## Files

- `_config.yml` - Jekyll configuration
- `_layouts/default.html` - Page template with styling
- `index.md` - Main content (edit this to update content)
- `photo.jpg` - Profile photo (replace with actual photo)
- `preview.html` - Standalone preview (not needed for deployment)

## Customization

Edit `index.md` to update content. The layout and styling are in `_layouts/default.html`.
